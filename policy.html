Privacy Policy

Last updated: April 25, 2026  ·  Effective immediately

1. Introduction

This Privacy Policy describes how PublicadorAutomatico, operated by Gustavo Bertazzoli ("we", "our", or "us"), handles information when you use our personal Pinterest automation application.

PublicadorAutomatico is a personal developer tool that uses the Pinterest API to publish and manage Pinterest content on behalf of the account owner. It is not a public-facing service and does not collect data from third-party users.

2. Information We Access

When authorised via the Pinterest OAuth flow, our application accesses:

  • Pinterest account information (username, profile data)
  • Pinterest boards and Pins belonging to the authorised account
  • Analytics data (impressions, saves, clicks) for published Pins

This data is accessed solely for the purpose of publishing, scheduling, and monitoring Pinterest content for the account owner. No data is sold, shared, or transferred to third parties.

3. How We Use Information

Information accessed through the Pinterest API is used exclusively to:

  • Create and publish Pins to designated boards
  • Read board structures for content organisation
  • Retrieve performance metrics for internal reporting
  • Manage OAuth tokens required for continued API access

4. Data Storage and Security

OAuth access tokens and refresh tokens are stored locally in a secure configuration file accessible only to the application operator. No Pinterest user data is stored in external databases or transmitted to third-party services beyond what is required to interact with the Pinterest API itself.

5. Third-Party Services

This application interacts with the Pinterest API v5. Use of the Pinterest API is governed by the Pinterest Developer Guidelines and Pinterest Terms of Service .

6. Your Rights (GDPR)

As a user based in the European Union, you have the right to access, rectify, or erase any personal data we hold. Since this is a personal tool with no external users, all data belongs to the operator. For any privacy-related enquiries, please contact us at the address below.

7. Changes to This Policy

We may update this Privacy Policy from time to time. The date at the top of this page reflects the most recent revision. Continued use of the application after changes constitutes acceptance of the updated policy.
